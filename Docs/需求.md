# 📱 lunadriod — 功能需求说明

用maui框架开发一个支持安卓手机和安卓电视的应用，所有的依赖包先检查nuget，下载最新的版本，UI框架使用最新的UraniumUI框架，应用能播放hls源的播放器，能下载hls链接到本地，项目分为Lunadriod.App，Lunadriod.Core，Lunadriod.Tests的结构，生成的项目存储在`D:\AndroidProject\Lunadroid`，生成过程存储在工程根目录的readme.md文件中

## 1. 🔐 Onboarding & Permissions

**1.1** 作为一个新用户，我希望先看到一个欢迎界面，接单介绍lunadriod是做什么的

**1.2** 作为一个新用户，我希望有一个简单的使用条款说明，并点击了同意按钮才能进行下一步

---

## 2. 📊 主页

**2.1** 作为一个用户，我希望有一个电影搜索界面，搜索的时候提示用户耐心等待，等待界面可以关闭，但是在搜索结果面板上要有提示，搜索还在进行中，在搜索界面面板上可以停止搜索。

**2.2** 作为一个用户，可以看到一个漂亮的影视列表，每个结果以卡片显示，其中包含宣传海报，电影名，电影打分，电影来源，满足鼠标浮上去的时候宣传海报放大1.05倍

**2.3** 作为一个用户，有一个搜索历史记录，可以单个删除，全部清除，选中一个历史记录可以触发搜索

**2.4** 作为一个用户，点开其中一个电影卡片后，可以看到详情，列出他的剧集列表，对这些剧集列表可以播放或下载，并记录在数据库中

**2.5** 作为一个用户，可以播放本地影视资源

---

## 3. 📋 历史

**3.1** 作为一个用户，可以看到一个历史播放记录和一个下载记录管理，每个记录

**3.2** 作为一个用户，可以按日，月，年分类，对每条记录显示影视名称，电影源，播放进度，观看时间，时间以刚刚，多少分钟前，多少小时前，多少天前，多少月前，多少年前这种时间分类

**3.3** 作为一个用户，可以删除其中一条，或全部清理

**3.4** 作为一个用户，选中一个记录可以选择播放，对下载的剧集要选择本地资源进行播放

**3.5** 作为一个用户，我可以搜索下载记录


---

## 4. 🏷️ 我的

**4.1** 作为一个用户，我可以管理电影播放资源，选中的才会被用于影视搜索

**4.2** 作为一个用户，可以添加新的电影播放资源，这些资源以影视源地址，影视源名称，影视baseurl，是否成人进行录入

**4.3** 作为一个用户，我也可以从云端"https://pz.v88.qzz.io/?format=0&source=full"录入影视资源

**4.4** 作为一个用户，可以看到这些资源的网络问题，包括能否访问以及访问耗时


---

## 5. 🔄 Sync & Data

**5.1** 作为一个用户，我希望所有的数据都存储在sqilte中，App配置文件在appconfig.json全局文件中

**5.2**  选择一个好用的orm库管理数据库

---

## 6. ⚙️ Settings

**6.1** 作为一个用户，我能切换主题配色

**6.2** 作为一个用户，我能清除所有应用数据，重置数据

**6.3** 作为一个用户，我能看到软件版本号

---

## 7. 🔒 Security

**7.1** As a user, I want to lock the app with a PIN or biometric (fingerprint) so others can't see my finances

**7.2** As a user, I want to set up app lock during onboarding and be able to disable it in settings
